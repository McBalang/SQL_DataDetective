SQL Beginner Lesson

SELECT statement:

The SELECT statement is used to retrieve data from one or more tables.

SELECT *:

The SELECT * statement is used to retrieve all columns from a table.

WHERE clause:

The WHERE clause is used to filter records based on a specified condition.

AND operators:

You can use AND combine multiple conditions. Using AND forces the result to satisfy both conditions

OR operators:

You can use OR to combine multiple conditions. Using OR will allow the result to satisfy either or both conditions

ORDER BY:

The ORDER BY clause is used to sort the result set in ascending (ASC) or descending (DESC) order. you can also sort the residents table in descending order

AND and OR operators:

You can use AND and OR to combine multiple conditions.
